# useHass: UseHassHook
`useHass()` returns high-level helper functions plus all data retrieved from the authenticated Home Assistant instance (entities, areas, floors, user info, etc.) via a single hook.

> Snapshot (`useHass.getState()`) is preferred for invoking static helper functions outside render. Subscribing to them provides no benefit and may cause unnecessary renders.

### Helpers API

The `helpers` object exposes utility methods that aren't strictly formatting, grouped by concern. See below for available helpers and usage:

- `helpers.dateTime.getTimeZone()` – resolved timezone after applying user preference (local/server) via `resolveTimeZone`.
- `helpers.dateTime.shouldUseAmPm()` – boolean indicating if 12h clock (AM/PM) should be used.

These pair with the granular formatter methods (`formatHour`, `formatMinute`, `formatSeconds`, `formatAmPmSuffix`) for building custom time displays. See the <Redirect to="/docs/core-hooks-usehass-helpers--docs">helpers</Redirect> documentation for more information.
