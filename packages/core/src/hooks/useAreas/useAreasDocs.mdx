# useAreas()
###### `useAreas(): Area[] | null`

A hook that will retrieve all the areas configured in home assistant and the related entities, services, floors and devices, this will return all information about the area like the name, picture and the area id.

The areas returned will match your areas in your Home Assistant instance.

This hook will return the data or null if not available yet.

- **Dependencies**: This hook requires you to use it within the `HassConnect` component.

<p>The following is the use of the hook:</p>
<Source dark code={`const areas = useAreas();`} />
