# useLocaleData()
###### `useLocaleData(): FrontendLocaleData | null`

A custom React hook to subscribe to the Home Assistant locale updates, this includes language, number and date formatting.

This hook will return the data or null if not available yet.

- **Dependencies**: This hook requires you to use it within the `HassConnect` component.
