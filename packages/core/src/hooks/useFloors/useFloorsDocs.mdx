# useFloors()
###### `useFloors(): FloorWithAreas[] | null`

A custom React hook to subscribe to the Home Assistant floor registry updates, and return a hierarchical list of floors and their associated areas, and the devices/entities within the areas, any areas not assigned to a floor will be part of the "Unassigned" floor.

The areas returned under each floor are returned in the same structure as the <Redirect to="/docs/core-hooks-useareas--docs">useAreas</Redirect> hook.

This hook will return the data or null if not available yet.

- **Dependencies**: This hook requires you to use it within the `HassConnect` component.
