# useUser()
###### `useUser(): HassUser | null`

A custom React hook to subscribe and retrieve the current authenticated user. This hook takes no parameters.

This hook will return the current user object or null if not available yet.

- **Dependencies**: This hook requires you to use it within the `HassConnect` component.
