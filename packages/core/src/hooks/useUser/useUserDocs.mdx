# useUser()
###### `useUser(): HassUser | null`

A custom React hook to retrieve the current authenticated user, this hook will not refresh unless you refresh the page or log back in, as the user typically does not change unless the user logs in or out.

This hook will return the current user object or null if not available yet.

- **Dependencies**: This hook requires you to use it within the `HassConnect` component.
