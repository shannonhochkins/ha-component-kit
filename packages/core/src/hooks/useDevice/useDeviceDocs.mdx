# useDevice()
###### `useDevice(entityId: EntityName): ExtEntityRegistryEntry | null`

The useDevice hook is designed to return all the additional information related to a device.

The above will either return null or the extended device information, this is not the entity, but rather the device, 
they're different in the eyes of home assistant, not all entities have devices.

- **Dependencies**: This hook requires you to use it within the `HassConnect` component.
